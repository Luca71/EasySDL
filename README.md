# EasySDL

## Game framework based on the following library:
- SDL2 - v2.0.22.0
- SDL2_image - v2.6.0.0
- SDL2_mixer - v2.6.1.0
- SDL2_ttf - v2.0.18.00

## Example project and usage: [NoonDuel - Two players C++ shooting game](https://github.com/Luca71/NoonDuel)
